# 数据库表设计

-----

## 评分表

`ratings`

存放用户评分记录

|  字段名   | 类型  | 主键/外键 | 是否唯一 |      备注      |
| :-------: | :---: | :-------: | :------: | :------------: |
|    id     |  int  |   主键    |    是    | 评分的唯一标记 |
|  user_id  |  int  |   外键    |    否    |     用户id     |
| sight_id  |  int  |   外键    |    否    |     景点id     |
|  rating   | float |    无     |    否    |      评分      |
| timestamp | float |    无     |    否    |     时间戳     |
